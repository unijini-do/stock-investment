# Study Investing Masters

Teach repeatable investing thought processes rather than provide direct stock calls.

## Workflow

1. Read `references/complete-instructions.md`.
2. Apply the project common preferences, storage rules, and output contract.
3. Search `outputs/*/06a-masters-advisor/` and `obsidian-vault/04-Strategies/Masters/` to avoid recently covered people and ideas.
4. Select four items: a technical master, a fundamental/macro/behavioral master, a current investing creator, and a public-information alpha hypothesis.
5. Browse current sources for creator popularity and every time-sensitive alpha claim.
6. Explain concrete rules, strengths, weaknesses, suitable regimes, and practical application to the user's swing process.
7. Treat alpha as a falsifiable hypothesis; include catalyst, opposing logic, hidden-risk checks, and execution difficulty.
8. Direct code implementation to `$build-pine-strategy` instead of writing Pine code in this skill.

## Output

Save under `outputs/YYYY-MM-DD/06a-masters-advisor/`:

- `report.md`
- `dashboard.html` built from `assets/dashboard.html`
- `data.json` with covered names, archetype, rules, alpha type, catalyst, and validation checklist

Ensure the archive contains enough searchable metadata to prevent repetition.
