# 분석 항목별 판단 기준

10개 항목 분석 시 적용하는 정량 기준과 해석 가이드. 절대선이 아니라 섹터·성장단계에 따라 조정한다.

## 3. 수익성 분석 (5년 추이)
- 추이의 **방향**이 절대 수준보다 중요. 마진이 우상향이면 가격결정력 강화 신호.
- 매출총이익률: 소프트웨어 70%+ vs 소매 20%대 — 섹터 기준으로 비교.
- 영업이익률 하락 + 매출 성장 = 성장 위한 투자인지, 경쟁 심화인지 구분.

## 4. 재무 건전성
| 지표 | 양호 | 주의 | 위험 |
|---|---|---|---|
| 부채비율(D/E) | < 1.0 | 1.0~2.0 | > 2.0 (자본집약 산업 제외) |
| 유동비율 | > 1.5 | 1.0~1.5 | < 1.0 |
| 현금 vs 총부채 | 현금 > 부채 | 비슷 | 부채 >> 현금 |
- 금융·유틸리티는 구조적으로 부채비율이 높음 → 섹터 표준으로 판단.

## 5. 잉여현금흐름(FCF)
- **FCF 수익률** = FCF / 시가총액. 4%+ 면 양호, 채권금리와 비교.
- FCF가 순이익을 꾸준히 상회 = 이익의 질 높음. 반대면 회계이익 의심.
- 자본배분 우선순위 파악: 재투자 / 자사주매입 / 배당 / 부채상환 / M&A — 트랙레코드로 경영진 평가와 연결.

## 6. 경쟁우위(해자) 점수 (각 1~10)
| 요소 | 9~10 | 5~6 | 1~2 |
|---|---|---|---|
| 가격결정력 | 인상해도 수요 유지 | 제한적 | 가격 수용자 |
| 브랜드 | 글로벌 프리미엄 | 지역 인지도 | 무차별 상품 |
| 전환비용 | 이탈 시 큰 비용/위험 | 다소 번거로움 | 즉시 대체 가능 |
| 네트워크효과 | 사용자↑ = 가치↑ 강력 | 약한 효과 | 없음 |
- 4개 평균이 7+ = 넓은 해자(wide moat), 4~6 = 좁은 해자, 4 미만 = 해자 없음.

## 7. 경영진 자질
- **자본배분 트랙레코드**: 과거 M&A·자사주매입이 주주가치를 창출했나 파괴했나.
- **내부자 지분**: 높을수록 이해관계 정렬(단, 지배구조 리스크도 점검).
- **보상 정렬**: 보상이 단기 주가가 아니라 장기 ROIC·FCF에 연동되는가.

## 8. 밸류에이션 스냅샷
- 현재 배수 vs **자사 5년 평균**: 프리미엄/디스카운트 판정.
- 현재 배수 vs **섹터 동종업체**: 상대 매력도.
- 고PER이 항상 비싼 게 아님 — 성장률(PEG), FCF 전환율과 함께 해석.
- EV/EBITDA는 부채 구조가 다른 기업 비교에 유용.

## 9. 강세/약세 시나리오 목표주가
- 강세: 낙관적 매출성장 + 마진확대 + 배수 재평가 가정.
- 약세: 성장둔화 + 마진압박 + 배수축소 가정.
- 각 시나리오의 **핵심 가정 2~3개**를 명시. 목표주가는 가정의 결과물임을 강조.

## 10. 결론 (확신 수준)
- **상(high)**: 해자 넓음 + 재무 건전 + 밸류 합리 + 촉매 명확.
- **중(medium)**: 일부 강점, 일부 불확실.
- **하(low)**: 논거는 있으나 변수 많음. 추가 관찰 권고.
