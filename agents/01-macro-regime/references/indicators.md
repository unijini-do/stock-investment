# 거시 지표 정의 · 검색 쿼리 · 해석 임계값

모든 수치는 최신 웹 검색으로 최신값 확인. 현재 연도를 쿼리에 정확히 반영.

## 후행지표 (현재 상태)

| 지표 | 검색 쿼리 | 해석 |
|---|---|---|
| 미 GDP | `US GDP growth rate latest quarter` | QoQ 연율 2%+ 견조 / 0~2% 둔화 / 마이너스 침체 |
| 한 GDP | `한국 GDP 성장률 최근 분기` | YoY 기준, 잠재성장률(약 2%) 대비 |
| 미 기준금리 | `Fed funds rate current` | 인상=긴축(성장주 부담) / 인하=완화(위험자산 우호) |
| 한 기준금리 | `한국은행 기준금리 현재` | 한미 금리차 계산용 |
| 미 10년물 | `US 10 year treasury yield` | 급등=멀티플 압박 / 하락=성장주 우호 |
| 미 CPI/PCE | `US CPI PCE inflation latest` | 목표 2% 대비, 근원(core) 중요 |
| 한 CPI | `한국 소비자물가 상승률 최근` | 한은 목표 2% 대비 |
| USD/KRW | `원달러 환율 현재` | 1300+ 원화약세(외국인 이탈) |
| DXY | `dollar index DXY` | 강세=신흥국·원화 약세 압력 |
| 미 실업률 | `US unemployment rate latest` | 급등=침체 신호 |

## 선행지표 (방향 예측 — 주식투자 핵심)

| 지표 | 검색 쿼리 | 해석 임계값 |
|---|---|---|
| **장단기 금리차 10Y-2Y** | `US 10Y 2Y treasury spread` | **역전(마이너스)=침체 선행 신호**, 정상화=회복 |
| **금리차 10Y-3M** | `10 year 3 month yield spread` | 연준이 선호하는 침체 예측 지표 |
| **ISM 제조업 PMI** | `ISM manufacturing PMI latest` | **50 기준선**: 위=확장, 아래=수축 |
| **ISM 서비스 PMI** | `ISM services PMI latest` | 서비스업 경기, 50 기준 |
| **신규 실업수당** | `US initial jobless claims latest` | 추세 상승=고용 둔화 조기 신호 |
| **하이일드 스프레드** | `US high yield credit spread` | 확대=리스크 회피, 축소=위험선호 |
| **VIX** | `VIX index today` | 20 이하 안정 / 30+ 공포 |

## 국면 판정에 쓰는 지표 묶음
- **회복**: 금리차 정상화 + PMI 50 회복 + 실업수당 고점 후 하락 + 금리 인하 사이클
- **확장**: GDP 견조 + PMI 55+ + 고용 강세 + 물가 상승 + 금리 인상
- **둔화**: PMI 하락(여전히 50+) + 금리차 축소/역전 + 물가 정점 + 고금리 부담
- **침체**: GDP 마이너스 + 실업률 급등 + 금리차 역전 후 정상화 + 금리 인하 전환

단일 지표로 판정하지 않는다. 묶음의 정합성으로 판정하고, 엇갈리면 전환 구간으로 표기.
