# 모니터링 소스 & 검색 패턴

데이터 수집 시 최신 웹 검색으로 접근한다. 각 소스별 검색 쿼리 패턴과 우선 수집 글 유형.

## 미국 커뮤니티

| 소스 | 검색 쿼리 패턴 | 우선 수집 글 유형 |
|---|---|---|
| r/wallstreetbets | `wallstreetbets DD <티커>`, `wsb trending` | 고추천 DD(심층분석) 스레드 |
| r/investing | `r/investing weekly discussion`, `r/investing <티커>` | 위클리·데일리 디스커션 |
| r/stocks | `r/stocks <티커> sentiment` | 종목 토론 스레드 |
| StockTwits | `stocktwits <티커> bull bear`, `stocktwits trending` | 트렌딩 티커 + Bull/Bear 비율 |
| X / Fintwit | `<티커> fintwit`, `$<티커> twitter sentiment` | $티커 멘션 톤 |
| Seeking Alpha | `seeking alpha <티커> comments` | 댓글 여론 |

추가: "AI 투자 프롬프트 모음", "투자정보 사이트 총정리" 류 인기글도 모니터링 (커뮤니티가 주목하는 도구·자료 파악).

## 한국 커뮤니티

| 소스 | 검색 쿼리 패턴 | 우선 수집 글 유형 |
|---|---|---|
| 디시 해외주식 마이너갤 | `디시 해외주식 갤러리 <종목>`, `해외주식갤 개념글` | 추천 많은 개념글(정보글) |
| 네이버 '미국주식이 미래다' | `미국주식이 미래다 <종목>`, `미국주식 카페 인기글` | 카페 인기글·등업글 |
| 클리앙 투자/금융 | `클리앙 투자 <종목>` | 투자/금융 게시판 인기글 |
| 유튜브 댓글 | `슈카월드 <주제> 댓글`, `박곰희TV <종목>` | 영상 댓글 여론 |

추가: "AI 투자 프롬프트", "미국주식 정보 사이트 정리" 류 인기글 모니터링.

## 감정 톤 정량화 힌트
- **미국**: StockTwits는 Bull/Bear 비율을 명시적 수치로 제공 → 그대로 활용. Fintwit은 멘션의 긍/부정 톤 추정.
- **한국**: 정량 수치가 드물다 → 댓글의 공포(손절·패닉·"망했다") vs 탐욕("가즈아"·"풀매수"·"이거 못사면 바보") 비중으로 추정.

## 데이터 신뢰도 주의
- 커뮤니티 데이터는 노이즈가 크다. 단일 글의 주장에 의존하지 않는다.
- 봇·작전·홍보글 가능성 항상 염두. 비정상적 동시 언급 급증은 인위적 펌핑 의심.
- 수집 실패 시 "해당 소스 접근 불가"로 명시하고 가용 소스만으로 분석 범위를 좁힌다.
